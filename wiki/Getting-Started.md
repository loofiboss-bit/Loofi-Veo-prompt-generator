# Getting Started

This page covers initial setup, first project creation, and first successful workflow pass.

## 1) Install

### Desktop

Download the latest release package for your operating system:

- Windows: NSIS installer or portable build
- Linux: AppImage
- macOS: DMG

### Source mode

```bash
git clone https://github.com/multidraxter-bit/Loofi-Veo-prompt-generator.git
cd Loofi-Veo-prompt-generator
npm install
npm run dev
```

For desktop runtime during development:

```bash
npm run electron:dev
```

## 2) First-time configuration

1. Open/create a project.
2. Set model and workflow defaults.
3. Configure API key preferences.
4. Enable or review autosave and export defaults.
5. Run onboarding/tutorial steps.

## 3) Build your first prompt cycle

1. Define subject, action, and environment.
2. Select style/camera characteristics.
3. Generate first prompt output.
4. Create one to three controlled variations.
5. Save preferred results to project history.

## 4) Build your first export

1. Select finalized prompt/history item.
2. Choose export format.
3. Validate metadata completeness.
4. Export and verify output file integrity.

## 5) Recommended initial habits

- Use templates for repeated job types.
- Keep prompt components modular.
- Save milestone snapshots before large edits.
- Maintain naming conventions for projects and presets.
