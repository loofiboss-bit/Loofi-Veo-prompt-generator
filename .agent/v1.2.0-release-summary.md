## 📦 What's New

### 🎨 Template System

Save and reuse your favorite prompt configurations! Create templates from your current setup, organize them with categories and tags, and share them with your team through import/export.

**Highlights:**

- One-click template creation
- Search and filter by category
- Import/export as JSON
- Built-in starter templates

### 🔤 Variable Placeholders

Make your prompts dynamic with variable placeholders! Use `{{variable_name}}` syntax to create reusable prompts that adapt to different scenarios.

**Example:**

```
A {{character_age}} year old {{character_gender}} wearing {{character_clothing}} 
walks through a {{location_name}} during {{time_of_day}}
```

**Highlights:**

- Auto-complete suggestions
- Built-in variables for character, location, time, camera, and style
- Custom variable creation
- Default value support: `{{variable:default}}`

### ⚡ Preset Management

Speed up your workflow with quick-apply presets! Choose from 10+ professionally crafted presets for camera work, lighting, style, and audio.

**Built-in Presets:**

- **Camera:** Cinematic, Action, Aerial Drone
- **Lighting:** Golden Hour, Film Noir, Neon Cyberpunk
- **Style:** Photorealistic, Anime, Vintage Film
- **Audio:** Cinematic, Documentary

**Highlights:**

- Favorites and recent tracking
- Custom preset creation
- Import/export presets

### 💾 Autosave & Recovery

Never lose your work again! Automatic saving with crash detection ensures your progress is always protected.

**Highlights:**

- Configurable autosave interval
- Crash detection on startup
- Recovery prompt with snapshot selection
- Last 5 versions saved
- Manual save points

### ⌨️ Keyboard Shortcuts

Work faster with 20+ keyboard shortcuts! Customize them to match your workflow and boost productivity.

**Popular Shortcuts:**

- `Ctrl+S` - Save project
- `Ctrl+Enter` - Generate prompt
- `Ctrl+Shift+E` - Export prompt
- `Ctrl+K` - Command palette
- `?` - Show shortcuts help

**Highlights:**

- Fully customizable
- Conflict detection
- Import/export shortcut profiles
- Context-aware shortcuts

### 📤 Enhanced Export

Export your work in multiple formats with improved reliability! Queue multiple exports and let the retry logic handle any issues.

**Supported Formats:**

- **JSON** - Structured data
- **PDF** - Formatted document with tables
- **CSV** - Spreadsheet compatible
- **Markdown** - Documentation format
- **XML** - Structured markup
- **ZIP** - Archive with multiple files

**Highlights:**

- Export queue with progress tracking
- Automatic retry on failure
- Batch export operations
- Export validation

### 🐛 GitHub Issue Templates

Help us improve Veo Studio! New issue templates make it easy to report bugs, request features, or suggest documentation improvements.

**Templates:**

- Bug Report (with severity levels)
- Feature Request (with priority)
- Documentation Request

## 🚀 Getting Started

### For Users

1. **Download the latest release** from GitHub
2. **Try the new features:**
   - Create a template from your current prompt
   - Use variables like `{{character_name}}` in your prompts
   - Apply a preset from the preset library
   - Press `?` to see all keyboard shortcuts

### For Developers

1. **Pull the latest changes:**

   ```bash
   git pull origin main
   git checkout v1.2.0
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start development:**

   ```bash
   npm run dev
   ```
