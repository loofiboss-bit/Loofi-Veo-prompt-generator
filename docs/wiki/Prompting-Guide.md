# Prompting Guide

Good Flow/Veo prompts are concrete, visual, and operational:

- Start with the subject and action.
- Lock character appearance when continuity matters.
- Describe camera movement and lens language.
- Include aspect ratio, duration, and resolution for API prompts.
- Add audio notes when timing or mood depends on sound.
- Use negative prompts for visual artifacts and unwanted objects.

Good Suno prompts separate style tags from lyrics. Avoid real artist names and
voice cloning requests.
