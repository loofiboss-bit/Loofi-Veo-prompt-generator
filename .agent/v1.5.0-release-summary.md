# v1.5.0 Release Summary (RC Stage)

**Version:** v1.5.0-rc.1  
**Date:** 2026-02-10  
**Type:** Release Candidate (Pre-release)

## Scope

This RC freezes the baseline v1.5.0 Performance & Stability scope for production readiness:

- Panel-level error boundary isolation
- Safe Mode startup guard in Electron
- Lazy loading for heavy overlays
