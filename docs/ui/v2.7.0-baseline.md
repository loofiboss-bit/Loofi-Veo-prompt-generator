# v2.7.0 UI Baseline

## Defect Matrix (Pre-overhaul)

| Area                 | Current Issue                                                                             | Impact                                                 | Priority |
| -------------------- | ----------------------------------------------------------------------------------------- | ------------------------------------------------------ | -------- |
| Theme                | Mixed theme systems (`body.light`, `data-theme`, startup script, inline component styles) | Inconsistent visual state and hard-to-reason overrides | P0       |
| Accessibility styles | Global rules force letter spacing, dialog positioning, touch target sizes for all users   | Core layout drift and visual regressions               | P0       |
| Modal stack          | Multiple modal implementations with inconsistent backdrop/z-index/focus behavior          | Overlay conflicts and unreliable close/focus handling  | P0       |
| Shell controls       | Floating help/settings FAB overlaps sidebar bottom navigation                             | Interaction and visual collision                       | P1       |
| Onboarding           | Tutorial step selectors mismatch current DOM attributes                                   | Steps fail or highlight wrong regions                  | P1       |
| Settings             | Modal and routed page duplicate styles/logic                                              | Drift, maintenance cost, inconsistent UX               | P1       |
