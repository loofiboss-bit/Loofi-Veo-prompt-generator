# v7.0.1 — Baseline Recovery

Released: 2026-07-10

v7.0.1 restores a dependable platform baseline before Creator Core (v8) expands
the production workflow.

## Highlights

- Retired Gemini models have been removed from user-selectable lists, pricing,
  tests, and executable fallback chains.
- Gemini 3.5 Flash is the recommended prompt and review default; Gemini 3.1 Pro
  and Flash-Lite remain explicit quality and economy choices.
- A typed lifecycle catalog and test prevent shut-down models from being added
  to production fallbacks.
- CI and development declarations use Node.js 24 LTS.
- Fedora/RHEL RPM metadata now uses native package names.

## Upgrade notes

Saved projects that reference a retired Gemini model are mapped to its supported
replacement only at execution time. Project data remains intact for the v8
model-preference migration.

## Validation

The release gate covers strict linting, type checking, unit tests, formatting,
production build, lifecycle checks, and Fedora RPM metadata inspection.
