# v7.0.0 — Director Mode

Released: 2026-07-10

Director Mode adds an approval-gated production workspace that carries a creative brief through
planning, Veo generation, structured review, revision, acceptance, and Creative Pack v2 export.

## Highlights

- Durable production runs survive reloads and offline interruptions.
- Every paid generation and structured review consumes an explicit approval allowance.
- Optional Gemini brief enhancement consumes a separate one-call approval before any cloud request.
- Known Veo operation IDs resume polling; ambiguous submissions require manual recovery.
- Completed media is cached in IndexedDB with provider expiry and media-risk tracking.
- Veo 3.1 capability rules and effective-dated resolution pricing are validated before approval.
- Accepted takes update assets, storyboard, and timeline together.
- Reviewers can explicitly reject, revise, or retake; each retry returns to approval preflight.
- Shots longer than eight seconds expose split and Veo-extension workflows instead of silent clipping.
- Creative Pack v2 includes run settings, provenance, reviews, and the timeline plan.

## Validation

The release gate covers unit, migration, recovery, route, accessibility, reload, and mocked
end-to-end Director workflows, followed by release validation and desktop packaging.
