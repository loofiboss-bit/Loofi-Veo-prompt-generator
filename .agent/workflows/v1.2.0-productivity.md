# v1.2.0 Productivity Layer Implementation Plan

**Version:** 1.2.0  
**Phase:** 2 of 5  
**Status:** In Progress  
**Target Date:** 2026-02-16

---

## Overview

Phase 2 focuses on enhancing user productivity through template management, presets, autosave recovery, keyboard shortcuts, and improved export reliability. This version builds upon the stable foundation of v1.1.0.

---

## Implementation Tasks

### 1. Template System (save/edit/delete)

**Goal:** Allow users to save, manage, and reuse prompt configurations as templates.

**Tasks:**

- [ ] Create `TemplateManager` service (`services/templateManager.ts`)
- [ ] Add template CRUD operations with IndexedDB persistence
- [ ] Create `TemplateLibrary` component for browsing templates
- [ ] Add template creation modal with name, description, and icon selection
- [ ] Implement template editing and deletion
- [ ] Add template import/export functionality (JSON)
- [ ] Create default starter templates (Cinematic, Documentary, Music Video, etc.)
- [ ] Add template search and filtering
- [ ] Update types to include `Template` interface with metadata

**Files to Create:**

- `services/templateManager.ts`
- `components/TemplateLibrary.tsx`
- `components/TemplateEditor.tsx`
- `components/TemplateCard.tsx`

**Files to Modify:**

- `types.ts` - Add Template interface
- `App.tsx` - Integrate template library UI
- `store/useAppStore.ts` - Add template state management

### 5. Keyboard Shortcuts

**Goal:** Comprehensive keyboard shortcut system for power users.

**Tasks:**

- [ ] Create `KeyboardShortcutManager` (`services/keyboardShortcutManager.ts`)
- [ ] Define global shortcut registry
- [ ] Implement shortcut conflict detection
- [ ] Create `ShortcutEditor` component for customization
- [ ] Add visual shortcut hints in UI
- [ ] Implement shortcut help overlay (`?` key)
- [ ] Add context-aware shortcuts
- [ ] Support shortcut sequences (vim-style)
- [ ] Add shortcut export/import for profiles

**Shortcuts to Implement:**

- `Ctrl+S` - Save project
- `Ctrl+Shift+S` - Save as template
- `Ctrl+N` - New project
- `Ctrl+O` - Open project
- `Ctrl+Enter` - Generate prompt
- `Ctrl+Shift+E` - Export prompt
- `Ctrl+K` - Quick command palette
- `Ctrl+/` - Toggle comments
- `Ctrl+D` - Duplicate current shot
- `Ctrl+Shift+D` - Delete current shot
- `F11` - Toggle fullscreen
- `?` - Show shortcuts help

**Files to Create:**

- `services/keyboardShortcutManager.ts`
- `components/ShortcutHelp.tsx`
- `components/ShortcutEditor.tsx`
- `hooks/useKeyboardShortcut.ts`

**Files to Modify:**

- `App.tsx` - Initialize shortcut manager
- `store/useSettingsStore.ts` - Store custom shortcuts
