# v1.2.0 Quick Reference Guide

Quick reference for all new productivity features in v1.2.0.

---

## 🎨 Templates

### Create Template

```typescript
import { createTemplateFromState, saveTemplate } from './services/templateManager';

const template = createTemplateFromState('My Template', 'Description here', promptState, {
  icon: 'template',
  category: 'cinematic',
  tags: ['action', 'outdoor'],
});

await saveTemplate(template);
```

### Load Templates

```typescript
import { getUserTemplates, getTemplate } from './services/templateManager';

// Get all templates
const templates = await getUserTemplates();

// Get specific template
const template = await getTemplate('template-id');
```

### Search Templates

```typescript
import { searchTemplates } from './services/templateManager';

const results = await searchTemplates('cinematic');
```

### Import/Export

```typescript
import { exportTemplates, importTemplates } from './services/templateManager';

// Export
const json = await exportTemplates(['id1', 'id2']);

// Import
const count = await importTemplates(json);
```

## 🔤 Variables

### Variable Syntax

```
{{variable_name}}                    // Simple variable
{{variable_name:default_value}}      // Variable with default
```

### Built-in Variables

```
{{character_name}}
{{character_age}}
{{character_gender}}
{{character_clothing}}
{{location_name}}
{{location_type}}
{{time_of_day}}
{{weather}}
{{camera_movement}}
{{camera_angle}}
{{art_style}}
{{color_palette}}
```

### Parse and Replace

```typescript
import { findVariables, replaceVariables } from './utils/variableParser';

// Find variables
const matches = findVariables('A {{character_age}} year old {{character_gender}}');

// Replace variables
const result = replaceVariables('A {{character_age}} year old {{character_gender}}', {
  character_age: '25',
  character_gender: 'woman',
});
// Result: "A 25 year old woman"
```

### Validate Variables

```typescript
import { validateVariables } from './utils/variableParser';

const { isValid, missing } = validateVariables(text, variables);
if (!isValid) {
  console.log('Missing variables:', missing);
}
```

### Custom Variables

```typescript
import { createCustomVariable } from './utils/variableParser';

const variable = createCustomVariable('my_variable', 'default value', 'Description of variable');
```

## ⚡ Presets

### Get Presets

```typescript
import { getAllPresets, getPresetsByCategory } from './services/presetManager';

// Get all presets
const presets = await getAllPresets();

// Get by category
const cameraPresets = await getPresetsByCategory('camera');
```

### Apply Preset

```typescript
import { getPreset } from './services/presetManager';

const preset = await getPreset('camera-cinematic');
if (preset) {
  // Apply preset.params to your state
  Object.assign(promptState, preset.params);
}
```

### Create Custom Preset

```typescript
import { createPreset, savePreset } from './services/presetManager';

const preset = createPreset(
  'My Preset',
  'Description',
  'camera',
  { cameraMovement: 'Dolly forward', cameraDistance: 'Wide shot' },
  { icon: 'video', tags: ['custom'] },
);

await savePreset(preset);
```

### Favorites

```typescript
import { addToFavorites, getFavoritePresets } from './services/presetManager';

// Add to favorites
await addToFavorites('preset-id');

// Get favorites
const favorites = await getFavoritePresets();
```

### Recent Presets

```typescript
import { trackRecentPreset, getRecentPresets } from './services/presetManager';

// Track usage
await trackRecentPreset('preset-id');

// Get recent
const recent = await getRecentPresets();
```

## 💾 Autosave

### Initialize

```typescript
import { initAutosave, markCleanShutdown } from './services/autosaveService';

// On app start
const wasCrashed = await initAutosave({
  enabled: true,
  intervalMs: 30000, // 30 seconds
  maxHistory: 5,
});

if (wasCrashed) {
  // Show recovery UI
}

// On app close
await markCleanShutdown();
```

### Mark Data as Dirty

```typescript
import { markDirty } from './services/autosaveService';

// When user makes changes
markDirty(currentState);
```

### Manual Snapshot

```typescript
import { saveSnapshot } from './services/autosaveService';

await saveSnapshot(currentState, 'Before major changes');
```

### Recovery

```typescript
import {
  getLatestAutosave,
  getAutosaveHistory,
  restoreFromSnapshot,
} from './services/autosaveService';

// Get latest
const latest = await getLatestAutosave();

// Get history
const history = await getAutosaveHistory();

// Restore
const data = await restoreFromSnapshot('snapshot-id');
```

### Control Autosave

```typescript
import { startAutosave, stopAutosave, forceAutosave } from './services/autosaveService';

// Start/stop
startAutosave(30000);
stopAutosave();

// Force immediate save
await forceAutosave();
```

## ⌨️ Keyboard Shortcuts

### Initialize

```typescript
import { initShortcuts, handleKeyboardEvent } from './services/keyboardShortcutManager';

// On app start
await initShortcuts();

// Add global listener
document.addEventListener('keydown', (e) => {
  handleKeyboardEvent(e);
});
```

### Register Shortcut

```typescript
import { registerShortcut } from './services/keyboardShortcutManager';

registerShortcut({
  id: 'my-action',
  name: 'My Action',
  description: 'Does something cool',
  keys: 'Ctrl+Shift+A',
  action: () => console.log('Action executed!'),
  category: 'general',
});
```

### Customize Shortcut

```typescript
import { customizeShortcut, resetShortcut } from './services/keyboardShortcutManager';

// Customize
await customizeShortcut('my-action', 'Ctrl+Alt+A');

// Reset to default
await resetShortcut('my-action');
```

### Get Shortcuts

```typescript
import { getAllShortcuts, getShortcutsByCategory } from './services/keyboardShortcutManager';

// Get all
const shortcuts = getAllShortcuts();

// Get by category
const editingShortcuts = getShortcutsByCategory('editing');
```

### Check Conflicts

```typescript
import { checkConflicts } from './services/keyboardShortcutManager';

const conflicts = checkConflicts();
if (conflicts.length > 0) {
  console.log('Shortcut conflicts:', conflicts);
}
```

## 📤 Export

### Quick Export

```typescript
import { quickExport } from './services/exportService';

await quickExport(data, 'json', 'my-export.json');
```

### Queue Export

```typescript
import { queueExport, getExportStatus, downloadExport } from './services/exportService';

// Queue export
const jobId = await queueExport('My Export', data, {
  format: 'pdf',
  filename: 'export.pdf',
  includeMetadata: true,
  retryAttempts: 3,
  onProgress: (progress) => console.log(`${progress}%`),
});

// Check status
const job = getExportStatus(jobId);

// Download when complete
if (job?.status === 'completed') {
  downloadExport(job);
}
```

### Supported Formats

```typescript
type ExportFormat = 'json' | 'txt' | 'pdf' | 'csv' | 'markdown' | 'xml' | 'zip';
```

### Validate Before Export

```typescript
import { validateExportData } from './services/exportService';

const { isValid, errors } = validateExportData(data, 'csv');
if (!isValid) {
  console.log('Validation errors:', errors);
}
```

## 🎯 Best Practices

### Templates

- Use descriptive names and descriptions
- Add relevant tags for easy searching
- Export templates regularly as backup
- Share templates with your team

### Variables

- Use consistent variable naming
- Provide default values when possible
- Document custom variables
- Validate variables before generation

### Presets

- Organize presets by category
- Use favorites for frequently used presets
- Create custom presets for your workflow
- Export presets for backup

### Autosave

- Set appropriate autosave interval (30-60 seconds)
- Review autosave history periodically
- Create manual snapshots before major changes
- Clear old snapshots to save space

### Keyboard Shortcuts

- Learn the most common shortcuts first
- Customize shortcuts to match your workflow
- Avoid conflicts with browser shortcuts
- Export your shortcuts for backup

### Export

- Validate data before exporting
- Use appropriate format for your needs
- Queue multiple exports for efficiency
- Keep export history for reference
