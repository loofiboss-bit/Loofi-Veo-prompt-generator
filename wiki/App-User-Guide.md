# App User Guide

This page is the practical, task-oriented user guide for Veo Studio.

Use this page when you need a fast path from first launch to a completed prompt workflow.

## 1) Before you start

- Install desktop release or run source mode from `README.md`.
- Open a project and confirm your API key configuration in Settings.
- Run onboarding once to learn shell navigation and help shortcuts.

## 2) Core daily workflow

1. Define the concept in Prompt workspace.
2. Select model/profile and tune style, camera, scene, and audio intent.
3. Generate baseline output.
4. Iterate with targeted variations (one major variable at a time).
5. Save approved results to history and templates.

## 3) Collaboration workflow (v3.5-v3.6)

1. Open **Collaborate** from the sidebar.
2. Complete **Profile Setup** if prompted.
3. Use **Comments** to capture review decisions.
4. Use **Roles** to control editing permissions.
5. Resolve state conflicts in **Conflict Resolution** before export.
6. Verify active users through the header **Presence** indicator.

## 4) Timeline and export workflow

1. Move approved prompt outputs into timeline/post workflow.
2. Build pacing and continuity with incremental edits.
3. Run a quick quality pass (audio levels, transitions, pacing).
4. Export in required format/profile and verify output integrity.

## 5) Troubleshooting quick map

- Prompt generation issues: check key validity, limits, and provider availability.
- UI performance issues: reduce concurrent heavy modules and restart runtime.
- Plugin issues: validate permissions and `engineVersion`, then retest baseline app.
- Desktop safe mode: inspect logs, disable risky modules, retry normal mode.
