# Installation and updates

Supported v8 desktop targets are Windows 11 x64 (NSIS and portable) and current Fedora x64 (RPM and AppImage). macOS remains experimental until signing and notarization are verified.

Updates come only from the allowlisted GitHub release channel. Metadata and SHA-256 must verify before installation. Unsigned artifacts are labeled community builds; signed builds include provenance, SBOM, manifest, and checksums.
