# Support

Use the public issue templates for bugs and feature requests:

https://github.com/loofiboss-bit/Loofi-Veo-prompt-generator/issues

For security reports, do not open a public issue. Follow [SECURITY.md](SECURITY.md).

Before opening a support issue, include:

- App version
- Operating system
- Install type
- Steps to reproduce
- Screenshots or logs when useful
