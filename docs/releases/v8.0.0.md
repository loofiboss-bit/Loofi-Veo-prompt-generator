# v8.0.0 — Creator Core

Released 2026-07-12.

Creator Core replaces the collection of disconnected specialist surfaces with one guided Create workflow while retaining those tools for advanced use.

## Product

- Six resumable steps: Plan, Scenes, Assets, Generate, Review, Export.
- Local deterministic planning and preflight run before any optional cloud review.
- Every automatic model decision shows its reason, lifecycle, constraints, maximum cost, and compatible fallback.
- Generation requires an explicit cost ceiling and persists operation acknowledgement before polling.
- A/B takes include video playback, review-dimension scores, notes, Keep, Reject, and Revise actions.
- Accepted video timing maps directly into complete Suno v5.5 and Studio 1.2 handoff briefs.

## Providers and trust

- Canonical model catalog with Gemini API and Vertex AI bindings, lifecycle metadata, capabilities, pricing dates, region restrictions, replacement/sunset state, and minimum SDK versions.
- Gemini API credentials remain in the OS vault; Vertex uses ADC/OAuth; Ollama is restricted to loopback hosts.
- Execution-aware retry and fallback only permit compatible availability/rate-limit fallbacks within the approved cost ceiling.
- Updates use the allowlisted GitHub channel and require checksum verification.
- Release automation generates SBOMs, checksums, manifests, provenance attestations, CodeQL, secret scanning, dependency review, and license checks.

## Desktop durability

- User-selected project storage with atomic downloads, SHA-256 metadata, asynchronous thumbnail/proxy generation, missing/corrupt detection, checksum relink, retention preview, and accepted-media protection.
- Durable paid-job recovery avoids duplicate charges after interruption.
- Versioned `.loofi-project` bundles include assets or portable references, provenance, catalog/pricing snapshot, checksums, and migration history.
- Automatic five-snapshot backups use atomic writes and checksum-verified restore.
- Diagnostics and redacted support bundles expose operational state without prompts or credentials; Safe Mode provides guided recovery.

## Compatibility

- Supported: Windows 11 x64 via NSIS/portable; current Fedora x64 via RPM/AppImage.
- Web/PWA remains a local-first fallback with browser storage and provider adapters.
- macOS remains experimental until signing, notarization, and CI verification are available.
- TypeScript remains on 5.9 for the v8 release because the current ESLint parser does not yet support TypeScript 7; the dependency graph otherwise installs cleanly without legacy peer flags.
