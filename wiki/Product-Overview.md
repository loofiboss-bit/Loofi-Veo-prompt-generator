# Product Overview

Veo Studio is a local-first AI video production workspace that combines structured prompt engineering, project orchestration, and editing workflows.

## Product mission

Enable creators and teams to turn narrative ideas into production-ready prompt and asset pipelines with consistent quality and repeatable process.

## Core value proposition

1. Keep creative and technical workflow in one environment.
2. Provide reliable local persistence and recoverability.
3. Make AI output generation auditable, iterative, and team-friendly.
4. Allow extension through a typed plugin platform.

## Primary user personas

| Persona            | Primary goals                         | Typical modules                            |
| ------------------ | ------------------------------------- | ------------------------------------------ |
| Solo creator       | Fast concept-to-output iteration      | Prompt, history, export, studios           |
| Creative lead      | Style consistency and review workflow | Project, templates, diagnostics, timeline  |
| Technical operator | Automation, reliability, scale        | Services, jobs, settings, CLI              |
| Plugin developer   | Extend product behavior safely        | Plugins, docs/PLUGIN_API, sandbox services |

## Capability map

| Capability area     | Included workflows                                          |
| ------------------- | ----------------------------------------------------------- |
| Prompt engineering  | Structured fields, presets, templates, model-aware tuning   |
| Asset continuity    | Project grouping, reusable settings, historical comparisons |
| Post-production     | Timeline, export, audio/video processing services           |
| Collaboration       | Shared session workflows and commentable review patterns    |
| Extensibility       | Plugin manifests, permissions, lifecycle hooks              |
| Desktop reliability | Safe mode, update channels, local diagnostics               |

## Module landscape

Feature modules currently include:

- `batch`, `collaboration`, `composer`, `diagnostics`
- `export`, `help`, `history`, `jobs`
- `onboarding`, `plugins`, `project`, `prompt`
- `settings`, `studios`, `timeline`, `workspace`

## Success metrics to track

- Time from concept to first viable prompt
- Iteration count to approved output
- Export success rate and retry rate
- Plugin health and activation stability
- Recovery success after crashes/safe mode
