# Google API key and provider profiles

- Gemini API is the individual default and stores its key in the operating-system credential vault on desktop.
- Vertex AI uses project, location, and Application Default Credentials/OAuth in Electron main.
- Ollama is loopback-only and supports local drafting/review, never paid Google video generation.

Use onboarding or Settings to test a connection. The test reports authentication, model availability, quota/rate-limit, region, and SDK hints without starting paid media generation.
