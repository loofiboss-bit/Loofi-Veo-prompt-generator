# Roadmap

## v6.0.0 — Creative Intelligence Workbench

- Promote optimization from side-panel assistance to a first-class `/optimize` workspace.
- Analyze prompt state, timeline shots, and assets through a local-first orchestration service.
- Add patchable suggestions with accept/dismiss history and undo-friendly prompt/timeline updates.
- Export one Creative Pack that combines Flow/Veo scene pack, Veo API prompt, Suno production brief, and timeline shot list.
- Replace public screenshot fixtures with real Playwright captures from seeded app state.
- Reduce release drift around update URLs, Vite documentation, i18n chunking, and export bundle size.

## v5.0.0

- Focus the app on Google Flow/Veo and Suno workflows.
- Add Flow scene packs, shot cards, compatibility scoring, and Veo API prompts.
- Expand Suno exports with production briefs and JSON output.
- Make Windows and Linux first-class release targets.
- Add public repo files, screenshots, wiki seed pages, and checksum generation.

## Later

- Deeper Flow/Veo automation once stable public APIs are available.
- More deterministic visual regression fixtures beyond public marketing screenshots.
- Broader packaging verification across Linux desktop environments.
- Optional macOS release artifacts when CI signing and verification are ready.
